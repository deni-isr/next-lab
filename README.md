## Setup Instructions

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Environment Variables**:
    Create a `.env.local` file in the root directory and add your credentials (see `.env.example` for reference).

3.  **Run in Development Mode**:
    ```bash
    npm run dev
    ```

4.  **Build for Production**:
    ```bash
    npm run build
    npm run start
    ```
